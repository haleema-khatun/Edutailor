# AI Career Readiness Platform

## Overview
This is a Flask-based AI web application that helps students analyze
their learning progress, interview performance, and resume ATS score.

## Features
- Skill Gap Analyzer using ML logic
- Interview Performance Analyzer using NLP
- Resume ATS Checker
- Dynamic analytics using Chart.js
- SQLite database for persistence

## Tech Stack
Backend: Python, Flask  
Frontend: HTML, CSS, Bootstrap  
AI/NLP: Scikit-learn, TextBlob  
Database: SQLite  
Visualization: Chart.js  

## How to Run
1. Create virtual environment
2. Install requirements
3. Run app.py
4. Open http://127.0.0.1:5000

## Use Case
- Final year project
- Hackathons
- Resume and portfolio
